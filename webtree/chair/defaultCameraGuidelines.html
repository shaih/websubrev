http://www.springer.de/comp/lncs/authors.html
Click on the link "Information for LNCS Authors", which will take you to a page with a zip file that includes the class file, some documentation on how to use that class file, and also the Springer copyright form. You must print and sign the copyright form, and then mail or FAX the signed copy to
Chair address here
and FAX number too

The page limit for your camera-ready paper is 18 pages total, including bibliography and all appendices. Please prepare an archive file (either zip, tar, tar.gz, or tgz) that includes the following files:

  • The source LaTeX file, including any figures, style files, and all other files that are needed to produce the camera ready copy.
  • A PDF file that was created from the source LaTeX file(s).
Then upload the archive file from the web-page at
[[[baseURL]]]/submit/cameraready.php
NOTE: you will need the submission-ID and password that were given to you when you first submitted the paper. Contact the chair if you lost your submission-ID and password. The deadline for uploading the files is [[[Camera-readery deadline]]], this is a firm deadline.
